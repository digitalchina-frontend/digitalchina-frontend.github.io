# VuePress

::: tip

什么是 VuePress？

VuePress 是静态网站生成器 :blue_heart:

:::

## 特点

- 为生成文档而 `生`

- Markdown 支持

  - [Markdown 拓展](https://vuepress.vuejs.org/zh/guide/markdown.html)

- SPA

- SEO 优化

- 懒加载页面

  - 显示加载进度

- Mobile 友好

  - Responsive

- i18n

## 核心功能

- 搜索

- ThemeConfig

  - sidebar
  - [yaml frontmatter](https://jekyllrb.com/docs/front-matter/)

- Vue 语法支持

  ```vue
  { { 1 + 2 } }
  ```

  ```vue
  <Hello>hi</Hello>
  ```

- 自定义主题
  - `meta data`
