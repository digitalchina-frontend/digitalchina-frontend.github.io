## 1. low code 介绍

### 1.1 介绍

- 低代码（Low-Code）是一种快速设计和开发软件应用程序并且手动编码最少的方法。它可以为技术人员提供更快，更可靠的价值。通过在图形界面中使用可视化建模来组装和配置应用程序，开发人员可以跳过所有基础架构让开发速度提升起来
